
・gitのインストールを確認する
git --version

・publicのプロジェクトをダウンロードする
git clone https://github.com/user/project.git


■登録する方法
git config --global user.name "Lin"
git config --global user.email "lin@example.com"

・gitにプロジェクトを追加する
git add git_lyph_public/


■ 1. 状態確認 
現在の状態を見る
git status
意味：
変更したファイル
add済み
commit待ち
push必要
■ 2. 変更内容確認
何を修正したか確認
git diff
add後を見る：
git diff --staged
■ 3. ファイル追加（ステージ）
変更をコミット対象にする
git add ファイル名
全部追加
git add .
■ 4. コミット
履歴に保存
git commit -m "修正内容"
例
git commit -m "login画面修正"
■ 5. リモート送信
GitHubへ送る
git push
初回：
git push -u origin main
GitHub に反映されます。
■ 6. 最新取得
他人の変更を取得
git pull
■ 7. クローン
最初に取得
git clone URL
例
git clone https://github.com/user/project.git
■ 8. ブランチ確認
現在のブランチを見る
git branch
■ 9. ブランチ作成
新しい作業用

git checkout -b feature01

新しい書き方

git switch -c feature01
■ 10. ブランチ切替
git switch main
または
git checkout main
■ 11. 変更取り消し
未addなら
git restore ファイル名
全部戻す
git restore .
■ 12. add取り消し
git restore --staged ファイル名
■ 13. 履歴確認
git log
短く見る
git log --oneline
■ 実務で一番使う4つ（重要）
特によく使うのはこれです。

git status
git add .
git commit -m "message"
git push
■ 作業の流れ
git status
git add .
git commit -m "修正"
git push
■ 覚え方
状態を見る
git status
登録する
git add .
保存する
git commit -m "内容"
送る


git status
git pull
git add .
git commit -m "コメント"
git push



| 操作        | コマンド                      | 説明               |
| --------- | ------------------------- | ---------------- |
| 状態確認      | `git status`              | 変更されたファイルを確認     |
| リポジトリ取得   | `git clone URL`           | GitHubなどからソースを取得 |
| ファイル追加    | `git add ファイル名`           | 指定ファイルをステージング    |
| 全ファイル追加   | `git add .`               | 変更ファイルをすべてステージング |
| コミット      | `git commit -m "コメント"`    | 変更内容を記録          |
| プッシュ      | `git push`                | GitHubへ送信        |
| 最新取得      | `git pull`                | GitHubから取得してマージ  |
| 取得のみ      | `git fetch`               | GitHubから取得のみ     |
| 履歴確認      | `git log`                 | コミット履歴表示         |
| 簡易履歴      | `git log --oneline`       | 1行で履歴表示          |
| 差分確認      | `git diff`                | 変更内容を確認          |
| ブランチ確認    | `git branch`              | ブランチ一覧表示         |
| ブランチ作成    | `git branch test`         | testブランチ作成       |
| ブランチ切替    | `git switch test`         | testブランチへ切替      |
| ブランチ作成＋切替 | `git switch -c test`      | 作成して切替           |
| マージ       | `git merge test`          | testブランチをマージ     |
| リモート確認    | `git remote -v`           | GitHub接続先確認      |
| ファイル削除    | `git rm ファイル名`            | Git管理対象から削除      |
| コミット取り消し  | `git reset --soft HEAD~1` | コミットのみ取り消し       |
| 強制的に戻す    | `git reset --hard HEAD~1` | コミットと変更を削除       |
